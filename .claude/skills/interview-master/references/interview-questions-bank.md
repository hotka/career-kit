# Interview Questions Bank

Comprehensive question library organized by role and difficulty level.

## Frontend Developer Questions

### Junior Level
1. What is the difference between `let`, `const`, and `var`?
2. Explain event bubbling and capturing
3. What are React hooks? Name 3 common hooks
4. How does CSS specificity work?

### Mid Level
1. Implement debounce and throttle functions
2. Explain virtual DOM and reconciliation
3. What are closures? Provide a practical use case
4. Describe state management patterns (Context, Redux, etc.)

### Senior Level
1. Design a component library architecture
2. Optimize React application performance (lazy loading, code splitting, memoization)
3. Implement server-side rendering strategy
4. Architecture patterns for large-scale frontend applications

## Backend Developer Questions

### Junior Level
1. What is REST? Describe HTTP methods
2. Explain MVC pattern
3. What is middleware in Express/Node.js?
4. Basic SQL queries (SELECT, JOIN, WHERE)

### Mid Level
1. Design RESTful API for e-commerce platform
2. Explain database indexing and optimization
3. Implement JWT authentication
4. Handle async operations and error handling

### Senior Level
1. Design microservices architecture
2. Database scaling strategies (sharding, replication)
3. Implement caching strategies
4. Design for high availability and fault tolerance
